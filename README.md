# Tetris Arcade

A retro-styled Tetris arcade experience built with React, TypeScript, Vite, and Tailwind CSS. 

## Features
- **Modern Responsive Layout**: Full-screen arcade cabinet style designed to dynamically adjust to various screen sizes.
- **Audio Sound Effects**: Handcrafted retro sound effects using the Web Audio API (Synthesizer) with built-in mute controls.
- **Vibrant Aesthetics**: High-fidelity dark mode with neon accents and micro-animations.
- **Offline Play & PWA**: Configured as a Progressive Web App (PWA) with offline capabilities.
- **Game Controls**: Includes support for both keyboard/touch controls and mobile haptic feedback.
